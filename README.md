# 🎬 WishFlix

Aplicação composta por uma **API em NestJS** e um **aplicativo mobile em Flutter** para gerenciar uma lista de filmes desejados.

---

## 🚀 Tecnologias

### Backend (NestJS)
- NestJS
- TypeScript
- Prisma (opcional)
- PostgreSQL

### Mobile (Flutter)
- Flutter
- Dart
- Material 3

---

## ▶️ Como iniciar o projeto

### Backend (NestJS)
```bash
cd backend
npm install
npm run start:dev
```

### Servidor iniciado em:
```
http://localhost:3000
```

### Mobile (Flutter):
```
cd mobile
flutter pub get
flutter run
```
